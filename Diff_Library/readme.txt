--윈도우
1. CMD창을 실행 프로젝트의 Diff_Library/bin/Release/net7.0/  경로로 이동
2. Diff_Library 입력 후 엔터
3. 실행


--리눅스
1. 윈도우에서 CMD를 실행.
2. CMD창에서 .csproj 파일의 경로(Diff_Library\)로 이동
3. dotnet publish -r linux-x64
4. 생성 완료된 <프로젝트>\linux-x64\publish 파일을 리눅스계열의 OS로 이동
5. 리눅스 터미널에서 dotnet Diff_Library.dll 입력 후 엔터
6. 실행
